Спеціальні мови програмування
ЛАБОРАТОРНА 2

Написати генератор що генерує множину А,
та знайде для цієї множини Б для всіх елементів множини що відповідають критерію В. 
Критерій В повинен бути оформлений у вигляді функції вхідний аргумент повинен бути скаляром а повертати функція повинна True або False.
Функція для перевірки критерію В, генератор для множини А, 
константи, та основна функція повинні знаходитись в різних файлах. 
Код повинен містити коментарі англійською мовою.

А для  n 1-15 - Арифметична прогресія з 100 елементів де два перших числа довільні числа вибрані студентом, записані в константи.
А для n 16-30 - Геометрична прогресія з 100 елементів де два перших числа довільні числа вибрані студентом, записані в константи.

Б для n n 1-5, 16-20 - Сума
Б для n 6-10, 21-25 - Добуток
Б для n 11-15, 26-30 - Середнє арифметичне

В для n 1, 6, 11, 16, 21, 26 - просте число
В для n 2, 7, 12, 17, 22, 27 - число є квадратом цілого числа
В для n 3, 8, 13, 18, 23, 28 - число є градусом по фаренгейту при якому вода знаходиться в рідкому стані, при тиску 760 міліметрів ртутного стовпчика.
B для n 4, 9, 14, 19, 24, 29 - сума цифрів числа цілочисельно ділиться на 3
B для n 5, 10, 15, 20, 25, 30 - цифри в числі розміщені в порядку зростання

Де n це порядковий номер у списку
